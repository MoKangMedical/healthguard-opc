# HealthGuard OPC

门诊人群健康管理平台 — Outpatient Care Health Management Platform

## 一句话定义

HealthGuard OPC 不卖门诊管理工具，卖门诊人群的健康管理结果。输入患者数据，输出风险分层+干预方案+效果追踪。

## 核心能力

- 风险分层: 多维度健康风险评估
- 干预方案: 基于指南的个性化干预
- 效果追踪: 自动化随访+效果评估
- MDT讨论: 多学科在线协作
- HIS对接: 模拟医院信息系统

## 快速开始

    git clone https://github.com/MoKangMedical/healthguard-opc.git
    cd healthguard-opc
    pip install -r requirements.txt
    python src/main.py
